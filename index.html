PhpBench

A Benchmarking Framework for PHP

What is PHPBench?

PHPBench is a micro/macro benchmarking framework for PHP, inspired by JMH and PHPUnit.

How does it work?

It repeats a method in benchmarking class and tells you the average time it takes to execute the method.

It also: performs multiple iterations, generates reports, provides statistics (including standard deviation and z-values), enforces stability, supports different time units and output modes and lots more.
